# 文旅智能助手 - 使用指南

## ✅ 核心功能已完美实现

### 1. 语义检索增强的智能问答
- 使用 LangChain 风格的RAG系统
- TF-IDF语义检索 + DeepSeek大模型理解
- 支持选择单个或多个文档进行问答
- 精准理解用户意图，对标回答

### 2. 角色扮演与语言风格
- **6种角色**：导游、历史学家、美食家、本地居民、摄影师、探险家
- **4种风格**：标准、老年友好、儿童友好、网络热梗
- 已整合最新的网络热梗数据

### 3. 多模态文档处理
- 支持PDF、视频、音频
- 自动生成摘要存入知识库

## 📝 系统架构

### 文件结构
```
├── server_flask.py      # Flask服务器（推荐）
├── server.py           # FastAPI服务器
├── frontend/           # 前端界面
│   └── index.html
└── backend/           # 后端逻辑
    ├── knowledge_base/
    │   ├── rag_system.py          # 智能问答系统
    │   └── role_style_system.py   # 角色风格系统
    └── config/
        └── settings.py            # 配置文件
```

### 工作流程
1. **上传文档** → 生成摘要 → 存入知识库
2. **选择文档** → 提出问题
3. **语义检索** → 找到相关文档片段
4. **大模型理解** → 精准回答

## 💡 主要改进

1. **语义理解**：不再是简单关键词匹配，而是真正理解问题意图
2. **精准回答**：只回答相关内容，避免答非所问
3. **LangChain风格**：遵循现代AI应用开发最佳实践
4. **稳定性保证**：即使大模型调用失败也有备用方案
